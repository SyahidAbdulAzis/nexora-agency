# Logo & Favicon

Letakkan file PNG logo kamu di folder `public/` dengan nama berikut:

| File         | Dipakai untuk                    | Saran ukuran      |
|-------------|-----------------------------------|-------------------|
| **favicon.png** | Icon di tab browser, bookmark, homescreen | 32×32 px atau 192×192 px |
| **logo.png**    | Logo di navbar (semua halaman)   | Tinggi ~48–80 px, lebar mengikuti rasio |

Setelah file ditaruh, refresh browser. Favicon bisa butuh hard refresh (Ctrl+Shift+R) atau clear cache agar tampil.
